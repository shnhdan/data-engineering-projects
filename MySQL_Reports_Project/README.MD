# MySQL Database + Reports Project

## Project Overview
Built a complete database system for an online store with automated SQL reporting and data visualization.

## Database Schema
- **products** - Product catalog with pricing and inventory
- **customers** - Customer information and contact details
- **orders** - Transaction records linking customers and products

## Features
- Relationaldatabase design with foreign keys \
- Complex SQL queries (JOINs, GROUP BY, aggregations) \
- Python-database integration \
- Automated report generation \
- Data visualization dashboard \
- Data quality validation 

## SQL Skills Demonstrated
- CREATE TABLE with constraints
- INSERT, SELECT, UPDATE
- INNER JOIN and LEFT JOIN
- GROUP BY with COUNT, SUM, AVG
- Date filtering and formatting
- Subqueries and aggregations

## Business Reports Generated
1. Sales by Category Analysis
2. Top Customers by Spending
3. Best Selling Products
4. Monthly Revenue Trends

## Tools Used
- SQLite (same syntax as MySQL)
- Python (sqlite3, pandas)
- Matplotlib for visualizations
